# Ejercicios de introducción

1. Mi primer script
2. Usar comentarios y variables
3. Lectura de datos introducidos por el usuario
4. Uso de comandos, como ls, en scripts
5. Un script que salude al usuario y le muestre la fecha y hora actual.
6. Un script que cree un nuevo directorio con un nombre proporcionado por el usuario.
7. Un script que muestre el directorio de trabajo actual y liste su contenido.
8. Calculadora Simple de Suma. Pide al usuario dos números, los guarda en variables y muestra el resultado de la suma. Este ejercicio refuerza el uso de read y la aritmética básica en Bash.
9. Verificador de Archivos o Directorios. Pide al usuario un nombre de archivo o directorio y comprueba si existe en la ubicación actual. Introduce el concepto de condicionales (if).
10. Organizador de Archivos. Crea un script que mueva todos los ficheros con una extensión específica (por ejemplo, .jpg) del directorio actual a una nueva carpeta llamada "Imagenes". Esto enseña el manejo de archivos y bucles para iterar sobre ellos.
11. Generar un número aleatorio entre 1 y 10. Luego, pedir al usuario que adivine el número. El script debe decirle si su intento es demasiado alto o demasiado bajo, hasta que acierte.
12. El script debe solicitar al usuario la ruta de un archivo. Luego, debe crear una copia de ese archivo en el mismo directorio, pero añadiendo la extensión .bak al final del nombre.
13. Modificar el script de "Hola, Mundo" para que, en lugar de pedir el nombre, lo reciba como un argumento al ejecutar el script. Por ejemplo: ./saludo_argumento.sh Ana. El script debería saludar a "Ana".
14. Escribir un script que cuente cuántos archivos (no directorios) hay en el directorio actual y muestre el resultado.
15. Crear un script que muestre un menú con tres opciones: 1) Listar el contenido del directorio, 2) Mostrar la fecha actual, 3) Salir. El script debe leer la opción del usuario y ejecutar la acción correspondiente hasta que elija salir.
16. Escribir un script que cree 10 archivos de texto vacíos con nombres secuenciales (por ejemplo, archivo1.txt, archivo2.txt, etc.) en el directorio actual.
